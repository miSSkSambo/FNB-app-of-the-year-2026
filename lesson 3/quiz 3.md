# Python Basics Quiz – Questions & Answers

## 1. A shopping application asks a customer to enter the quantity of items they want to purchase. Which line of code ensures the quantity can be used in a calculation?

### Options
- `quantity = str(input("Quantity: "))`
- `quantity = input("Quantity: ")`
- ✅ `quantity = int(input("Quantity: "))`
- `quantity = type(input("Quantity: "))`

**Answer:** `quantity = int(input("Quantity: "))`

---

## 2. What does `abs(-15.5)` return?

### Options
- `-15.5`
- `15`
- ✅ `15.5`
- `-15`

**Answer:** `15.5`

---

## 3. What will happen when the following code is executed?

```python
age = input("Age: ")
print(age + 5)
```

### Options
- It displays the user's age plus 5.
- It displays the age as a decimal.
- ✅ A `TypeError` occurs.
- Python automatically converts the input to an integer.

**Answer:** A `TypeError` occurs.

## 4. A game awards one bonus point for every 10 coins a player collects. A player has collected 47 coins. Which expression calculates the number of bonus points earned?

### Options
- `47 % 10`
- ✅ `47 // 10`
- `47 / 10`
- `47 ** 10`

**Answer:** `47 // 10`
