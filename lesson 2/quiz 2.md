## Question 3

**Which of the following correctly creates an f-string?**

- A. `'Hello {name}'`
- B. `f'Hello {name}'`
- C. `f'Hello (name)'` ✅
- D. `format('Hello', name)`

**Answer:** **`f'Hello {name}'`**

---

## Question 4

**Which code correctly displays a person's name using an f-string?**

- A. `print("Welcome {name}")`
- B. `print(f"Welcome, {name}")` ✅
- C. `print("Welcome," + {name})`
- D. `print(f{Welcome, name})`

**Answer:** **`print(f"Welcome, {name}")`**
